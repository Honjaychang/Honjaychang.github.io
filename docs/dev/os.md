# 操作系统

#### 进程线程

- 进程是资源分配的最小单位，线程是CPU调度的最小单位

进程

- 进程负责为程序的运行提供必备的环境
- 进程就相当于工厂中的车间线程

线程

- 线程计算机中的最小的计算单位，线程负责执行进程中的程序

- 线程就相当于工厂中的工人
